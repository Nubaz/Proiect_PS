Repartiția Bernoulli

Repartiția Bernoulli este un tip de repartiție discretă de probabilitate pentru variabile aleatoare discrete.

Ea figurează cele două rezultate posibile ale unui eveniment de tip boolean: succes și eșec, da și nu, adevărat sau fals etc.

Repartiția Bernoulli este un caz parametrizat al repartiției binomiale, pentru un singur experiment.

Media este egală cu probabilitatea p.

Dispersia este egală cu produsul probabilităților celor două stări ale evenimentului.

Funcția de masă de probabilitate

Funcția sa de masă are forma generală:

$$Pr(X=k)=\left\{ \begin{array}{ll} p &\mbox{, } x=1 \\ q=1-p &\mbox{, } x=0\end{array} \right.$$

Funcția de repartiție

Funcția sa de repartiție are forma generală:

$$Pr(X\le k)=\left\{ \begin{array}{ll} 0 &\mbox{, } x<0 \\ 1-p &\mbox{, } 0\le x<1 \\ 1 &\mbox{, } x\ge1\end{array} \right.$$
