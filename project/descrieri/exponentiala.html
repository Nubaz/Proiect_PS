Repartiția exponențială

Repartiția exponențială este un tip de repartiție continuă de probabilitate pentru variabile aleatoare continue.

Ea reprezintă distribuția probabilității timpului scurs dintre un set de evenimente, ce se manifestă independent cu frecvență o λ. Atât această frecvență, cât și natura evenimentelor, sunt identice cu cele descrise în cadrul repartiției Poisson.

Parametrul λ trebuie să fie întotdeauna pozitiv.

Valoarea sa modală este 0.

Funcția de densitate de probabilitate

Funcția sa de densitate are forma generală:

$$f(x)=\left\{ \begin{array}{ll} \lambda e^{-\lambda x} &\mbox{, } x\ge0 \\ 0 &\mbox{, } x<0\end{array} \right.$$

Funcția de repartiție

Funcția sa de repartiție are forma generală:

$$F(x)=\left\{ \begin{array}{ll} 1-e^{-\lambda x} &\mbox{, } x\ge0 \\ 0 &\mbox{, } x<0\end{array} \right.$$
