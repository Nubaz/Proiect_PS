Repartiția uniformă

Repartiția uniformă este un tip de repartiție continuă de probabilitate pentru variabile aleatoare continue.

Repartiția descrie rezultatul unui eveniment, ce se poate afla între două valori ale unui interval, una minimă (a) și una maximă (b).

Funcția de densitate de probabilitate

Funcția sa de densitate are forma generală:

$$f(x)=\left\{ \begin{array}{ll} \frac{1}{b-a} &\mbox{, } x\in (a,b) \\ 0 &\mbox{, } \text{în rest} \end{array} \right.$$

Funcția de repartiție

Funcția sa de repartiție are forma generală:

$$F(x)=\left\{ \begin{array}{ll} 0 &\mbox{, } x<{a} \\ \frac{x-a}{b-a} &\mbox{, } x\in(a,b) \\ 1 &\mbox{, } x>{b} \end{array} \right.$$

Propietăți speciale

Media și mediana repartiției au aceeași valoare, identică cu mijlocul intervalului (a,b), iar valoarea sa modală se poate afla oriunde în cadrul acestui interval.
